Import your own data for calculation

If you want to import your own data in ThermoBar-Hub you will need to format them accordingly to the provided templates.

You can download these templates in the list below:

  • Olivine - Liquid : here

  • Feldspar - Liquid : here

  • Alkali feldspar - Plagioclase : here

  • Clinopyroxene - Liquid : here

  • Clinopyroxene - only : here

  • Orthopyroxene - Liquid : here

  • Orthopyroxene - only : here

  • Orthopyroxene - Clinopyroxene : here

  • Amphibole - Liquid : here

  • Amphibole - only : here

  • Amphibole - Plagioclase : here

These templates work as it follow:

  • Each column correspond to the concentration (in wt.%) of an oxyde in the considered phase as "(oxyde)_(phase)" for example, concentration of Na2O in Clinopyroxene is given as "Na2O_Cpx"
  • You can keep your sample name by adding a "Sample_ID_(phase)" column.

You can add whatever column you want but if you want to keep them paired with your samples don't forget to add "..._(phase)" after its name. (example: "Rock type" become "Rock type_cpx")

Warning

Decimals must be separated using period/full-stop (.) and not comma (,)

Pay attention to remove all special characters or misplaced spaces (eg.: before values as "_32.4" instead of "32.4")
